# **App Name**: Coming Soon Landing

## Core Features:

- Title Display: Display a title indicating that the page is under development.
- Subtitle Display: Show a subtitle clarifying that the page will be available soon.
- Subtle Animations: Implement subtle animations using GSAP for a visually appealing loading experience.
- Responsive Design: Ensure the landing page is fully responsive across various screen sizes.
- Automatic Theme: Implement automatic dark/light mode based on the user's system preferences using prefers-color-scheme.
- Animated Background: Use CSS keyframes to create animated blobs/particles in the background with subtle movement.
- SEO setup: Make a basic accesible SEO setup. Semantical elements. Descriptive title tag

## Style Guidelines:

- Primary color: Zodiac Blue (#006AFF) for a modern and trustworthy feel.
- Background color: A very light desaturated variant of Zodiac Blue (#E5F0FF), providing a calm backdrop.
- Accent color: Atoll (#00ADA0), offering a contrasting but harmonious highlight for interactive elements.
- Font: 'Inter', a grotesque-style sans-serif, will be used for both headlines and body text, ensuring a modern and clean look. Note: currently only Google Fonts are supported.
- Content will be centered both vertically and horizontally, providing a clean and focused user experience. Fluid margins and paddings will adapt to different screen sizes.
- GSAP will be used for subtle animations, such as fading in the title and subtitle. These animations will respect the user's 'prefers-reduced-motion' setting.
